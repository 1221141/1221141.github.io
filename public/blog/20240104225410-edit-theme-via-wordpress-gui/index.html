Edit theme via wordpress gui

/linux/ /wordpress/

2024-01-04T22:54:10

In order to edit a theme in the wordpress gui go to Appearance > Theme File Editor.

In order to have an update button the correct used should be www-data. To fix this go to the wp-content page for your theme and run this.

chown www-data:www-data  -R * # Let Apache be owner
find . -type d -exec chmod 755 {} \;  # Change directory permissions rwxr-xr-x
find . -type f -exec chmod 644 {} \;  # Change file permissions rw-r--r--
