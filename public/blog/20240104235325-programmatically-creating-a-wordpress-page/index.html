I recently learned that WordPress provides a JSON API. This way I could consume and also post new content via API.

Might want to investigate this to help me when I want to track movies that I watched. Maybe a bookmarklet from IMDb or tmdb that would create the page for me.

Example in this video:

https://youtu.be/_w4Ok-lI48g?si=hA9iUsj2z2oAjh6w

I found another way to do that.

  1. add a php file in the wordpress root
  2. this file is now available as wp-ip/file.php
  3. pass the parameters in a get request and use the wp_insert_post to create a post programatically.
  4. might be more unsafe, but ok for a offline blog.
